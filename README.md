# Fast Blog — A FastAPI Project

## Overview

This project is a lightweight web application built with **FastAPI**, **HTML**, and **CSS**, designed to explore modern Python backend development and clean UI structure. It demonstrates API routing, template rendering, and modular project organisation suitable for small to medium‑sized applications.

## Features

-   FastAPI backend with async route handling
    
-   Jinja2 templating for dynamic HTML pages
    
-   Modular folder structure for scalability
    
-   Static assets (CSS, images) served efficiently
    
-   Developer‑friendly environment with auto‑reload
    

## Tech Stack

-   **Backend:** FastAPI, Python 3.x
    
-   **Frontend:** HTML5, CSS3
    
-   **Server:** Uvicorn
    
-   **Tools:** VS Code, Git, GitHub
